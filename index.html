sequence

a step sequencer with a chord-relative harmonic engine. set a root and a scale, author chords by degree — the rest of the band harmonizes against whatever played last.

02 / texture

texture

a loop mangler in the browser — drop any audio file, select the slice you want, pull it apart: vari-speed tape, a granular cloud, a self-oscillating noise filter.

print the result as a longer texture wav — one full pass at whatever speed you've bent it to.
