i've been building out this software for a month or so now and it's starting to feel less like a toy and more like something that could practically be used. after a few conversations with some friends i respect deeply the biggest takeaway was that this needs to be able to exist and live within a live show context - tools for the studio are fun but if i'm going to build a band around this concept it needs to be able to exist live and not in a "i exported all the stems into ableton" kind of way. 

this is the first test of attempting to make this really work "live" — everything is sequenced via the application, midi feeding out to external instruments, clock feeding out to PJ on the drums. we recorded the initial take live and then picked up some misc shots to make this less totally boring for y'all to look at.

bonus - features a ton of visual overlays also created via the ns-visual system
